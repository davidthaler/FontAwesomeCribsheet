Using Font Awesome

Setup

To use the Font Awesome icon library, you first need to get an account at Font Awesome. The free account should be fine. Once you get your account, you can access the library via a download or a CDN.

When you sign-up a Font Awesome "kit" will be created for you. To use the CDN to import Font Awesome, include a tag like this in the <head> of your page:

<script src="https://kit.fontawesome.com/[your kit name].js" crossorigin="anonymous"></script>

To download Font Awesome and serve it locally, go to the Font Awesome self-hosting page to get the download. Then link the stylesheet in like you normally would:

<link href="/your-path-to-fontawesome/css/all.css" rel="stylesheet">

Basic Usage

Once the Font Awesome library is included in your page, you just set the classes on any element to some of the Font Awesome classes. For example:

<span class="fas fa-arrow-left"></span>

gives us a left arrow:

Usually, you will need to specify 2 or 3 classes. The first is one of far|fas, this specifies a solid or regular (outline) version of the icon. The second class has the form fa-[Icon Name] and specifies the icon. There may also be a sizing class, which is explained in the section below.

Sizing

The icons below all have class="fas fa-search". The icon size can be adjusted by adding a Font Awesome class that controls the size to the class list.

  • "fa-xs"
  • "fa-sm"
  • None, or "fa-md"
  • "fa-lg"
  • "fa-2x"

Styling

The Font Awesome icons can be sytled in CSS like text. So the code:

<span class="fas fa-check-square fa-lg" style="color:green"></span>

gives us a green checkmark:

You can also change the icon size in CSS with the font-size property. For example, the code:

<span class="fas fa-search" style="font-size: 3rem;"></span>

Gives us a large search icon:

Lists and Fixed-Width Icons

Icons have different widths, which can be a problem if they are stacked vertically. Font Awesome includes a class fa-fw for fixed-width icons that will form a nice column when stacked vertically. You can read more about fixed-width icons in the Font Awesome fixed-width documentation page.

The Useful Icons section below does not use fixed-width icons. Instead, it uses the Font Awesome list icon classes fa-ul on the list itself, and fa-li on each list item.
